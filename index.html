Tentang Saya

Profil dan pendidikan saya

Saya adalah siswa SMK BPP Bandung jurusan Rekayasa Perangkat Lunak (RPL). Selama pendidikan, saya telah belajar tentang mengembangkan website fungsional menggunakan teknologi seperti HTML, CSS, JavaScript, PHP, dan MySQL, sekaligus merancang antarmuka yang menarik dan user-friendly.

Pengalaman saya sebagai Ketua Divisi PDD di OSIS telah membentuk kemampuan kepemimpinan dan manajemen proyek yang kuat. Saya memimpin tim untuk menciptakan berbagai materi desain grafis seperti poster, spanduk, dan konten digital untuk mempromosikan kegiatan sekolah, serta bertanggung jawab dalam dokumentasi visual seluruh acara penting.

Saya memiliki passion yang besar terhadap teknologi web dan seni digital, dan selalu bersemangat untuk mempelajari framework, tools, dan metodologi baru dalam pengembangan software.
